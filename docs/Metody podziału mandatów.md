# Metody podziału mandatów
Diego proponuje co najmniej pokrycie następujące metody
- Highest averages:
    1. [Metoda D'Hondta](https://en.wikipedia.org/wiki/D%27Hondt_method) (używana w Polsce)
    2. [Metoda Sainte-Laguë](https://en.wikipedia.org/wiki/Sainte-Lagu%C3%AB_method)
- Uwzględniające część ułamkową przydziału:
    1. [Metoda Hamiltona](https://math.libretexts.org/Bookshelves/Applied_Mathematics/Math_in_Society_(Lippman)/04%3A_Apportionment/4.02%3A_Hamiltons_Method)
    2. [Metoda Lowndesa](https://math.libretexts.org/Bookshelves/Applied_Mathematics/Math_in_Society_(Lippman)/04%3A_Apportionment/4.06%3A_Lowndes_Method)
- Modyfikujące dzielnik ([omówienie](https://jamescurley.blog/posts/2020-10-16-apportionment-methods/)):
    1. Metoda Jeffersona
    2. Metoda Adamsa
    3. Metoda Webstera
    4. Metoda Deana
- Trudno sklasyfikować:
    1. [Metoda Huntingtona-Hilla](https://en.wikipedia.org/wiki/Huntington%E2%80%93Hill_method)
